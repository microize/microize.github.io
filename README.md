# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, showcasing my professional experience, skills, projects, and certifications.

## Features
- **About Me**: Highlights my expertise in Azure Data Engineering and Generative AI.
- **Portfolio**: Displays projects across various categories like Data Engineering, Machine Learning, and Design.
- **Resume**: Downloadable resume for detailed professional information.
- **Contact**: Interactive contact form integrated with Formspree.

## Technologies Used
- HTML5, CSS3, JavaScript
- Google Fonts, Ionicons
